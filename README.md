# WeatherScope

A modern weather dashboard built with vanilla JavaScript.

## Features

- Current weather
- Hourly forecast
- 5-day forecast
- Air Quality Index
- UV Index
- Weather map
- City autocomplete
- Favorite cities
- Recent searches
- Dynamic weather backgrounds
- Dark / Light mode
- Responsive design
- Progressive Web App (PWA)
- Keyboard shortcuts
- Skeleton loading
- Lighthouse 100/100/100/100

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES Modules)
- OpenWeather API
- OpenStreetMap
- Leaflet.js
- Vercel
